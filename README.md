### Description

## Project Information

### How to Run
1. Clone the repository by selecting the green **code** button in GitHub, and copying the repository link.

2. Open a terminal and clone this repository to your local machine using the command:<br>
   ```bash
   git clone https://github.com/TomaszP05/Linimate.git

3. Navigate to the project folder and open `animate.html` in your preferred browser.

### Controls

- Pause & Play animation: `K`
- Reset animation: `R`
- Room in & out: `+` & `-`
- - Alternatively `scroll`
- Camera pan: `←`, `↑`, `→`, `↓`
